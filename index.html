HTML

  • The head element contains information about the workpage
  • The body element is known as the parent element
  • Elements written in the body element are known as child elements
  • The body element represents the visible content shown to the user.
  • The title element is used for a title to a webpage
  • Metadata can be describes as descriptions, keywords, author, etc.
  • The base element is used for a URL for a webpage

CSS

  • The border property assigns a border with the width in the value property
  • The padding property adds space around a content inside an element
  • The margin property indicates how much space we want around the outside of an element
  • The declarations in the CSS rule contain the CSS property that will be applied plus the value of the property
  • An example of a CSS rule could be adding "color" as the CSS property and "blue" would be the value of the property
  • A "link" element creates a connection to an external file

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m commits your staged content as a new commit snapshot
  • gitbash is a terminal that you input all your commands
  • when using gitbash, typing "cd" with a directories name is how to access a directory
  • git branch list all the branches that are currently on the computer that you're using
  • git checkout main switches back to the default (main) branch

JavaScript

  • A function is a set of instructions that tells the computer how to perform a certain task
  • A variable refers to a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes a code
  • "if" statements will only execute when a condition is met or is true
  • A strict equality operator "===" checks to see if two values are equal
  • An array is a single variable that's used to hold a group of data
  • An index is a unique number that identifies a location of the data item in the array
  • The first index in Javascript will always be 0
